### 课程介绍

在本课程中，我们将带领您学习如何使用Go语言从零开始构建一个功能齐全的Web框架。我们将一步步地深入讲解各个关键组件，并且通过实战项目，让您快速掌握Go语言编写Web框架的技巧。本课程适合有一定编程基础并且希望了解Go语言Web开发的学员。

### 课程大纲
1. 一个简单的http服务
2. http源码解读
3. 实现自定义handler
4. 包装 context
5. 实现c.Json 和 c.String
6. 包装run方法
7. 重构代码
8. 实现post方法
9. 实现路由组功能
10. 动态路由匹配
11. 中间件
12. 实现 c.BindJson
